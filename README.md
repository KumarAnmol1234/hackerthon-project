This project is a student-focused finance tracker web application designed to help users manage their monthly expenses in an organized and visual manner.

The user starts by entering their monthly/daily expenses under different categories such as food, travel, education, and entertainment. All data is securely stored in a database.

The application automatically calculates total expenses and presents the data through interactive graphs like pie charts and bar/line charts. These visual insights help students clearly understand their spending patterns, control unnecessary expenses, and develop better financial habits.
